All files can be run using "python3 %name_file%.py"

The naive approach using the global average rating can be run using avg_rating.py
The naive approach using the movie average rating can be run using avg_rating_movie.py
The naive approach using the user average rating can be run using avg_rating_user.py
The naive approach using the linear combination of the averages can be run using lin_regress.py

The matrix factorization is implemented in matrix_factorization2.py. The parameters are initialized at the start of the document and can be changed to correspond with the different experiments we ran using matrix factorization.

The folder datasets containing the data should be put into the same folder as the code.
